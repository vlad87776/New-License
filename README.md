# APEX (Author’s Permissive Evaluation eXcript)

**APEX** — это авторская лицензия, созданная для защиты свободы программного обеспечения и обеспечения доступности кода для всех пользователей[cite: 1, 2].

## Основные принципы (Key Principles)

* **Свобода модификации:** Вы имеете полное право изменять и адаптировать Программу под свои нужды[cite: 2, 21].
* **Copyleft (Наследование):** Любые измененные версии должны распространяться на условиях этой же лицензии (APEX)[cite: 11, 31].
* **Прозрачность:** При передаче модифицированного кода необходимо указывать уведомления об изменениях и их датах[cite: 10, 30].
* **Безотзывность:** Права предоставляются на весь срок действия авторского права при соблюдении условий[cite: 8, 27].

## Использование (Usage)

Чтобы использовать эту лицензию в своем проекте, просто добавьте файл `APEX.LICENSE` в корневой каталог вашего репозитория.
